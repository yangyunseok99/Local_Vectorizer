Figma 목업 제작 튜토리얼

  1. 모드 선택: 직각 UI를 선택합니다.
  2. 노이즈 제거: 슬라이더를 우측으로 당겨 자잘한 글씨나 아이콘이 사라지게 만듭니다. (덩어리만 남김)
  3. 복사 및 붙여넣기: 코드를 복사하여 Figma 캔버스에 Ctrl+V 합니다.
  4. 마무리: 깔끔하게 따진 버튼/카드 형태 위에 Figma 텍스트 툴(T)을 이용해 진짜 글씨를 타이핑합니다.

프리셋 설명

  • 직각 UI: 곡선을 강제로 펴서 반듯한 사각형(버튼, 카드)을 만듭니다. 블러 처리가 없어 선명합니다.
  • 둥근 UI: 둥근 버튼(Border-radius)이나 부드러운 UI 요소를 추출할 때 사용합니다.
  • 일러스트: 형태를 강제 변형하지 않고 원본의 디테일(곡선, 복잡한 형태)을 최대한 유지합니다.
