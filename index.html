캡처한 UI 올리기

따라 만들고 싶은 앱 화면이나 아이콘을 드래그해서 올려주세요. AI가 디자인 구조를 열심히 분석할 거예요.

2

복사 버튼 누르기

변환이 완료되면 우측 아래의 복사하기 버튼을 눌러주세요. 피그마가 이해할 수 있는 코드로 클립보드에 담겨요.

3

피그마에서 마법 확인하기

Figma를 켜고 빈 캔버스에 Ctrl + V (맥은 Cmd + V)를 눌러보세요. 글씨도 바꿀 수 있고 색상도 바꿀 수 있는 진짜 UI 레이어가 생겨나요!
