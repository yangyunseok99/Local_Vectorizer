이용 안내

Figma에서 사용하는 방법

  1. 결과물 우측의 클립보드 복사 버튼을 클릭합니다.
  2. Figma 프로그램을 열고 작업 중인 디자인 파일을 엽니다.
  3. 빈 캔버스를 클릭하고 Ctrl + V (Mac은 Cmd + V)를 누릅니다.
  4. 벡터 레이어로 완벽하게 붙여넣어지며, 자유롭게 색상 변경 및 크기 조절이 가능합니다.

✨ 어떤 이미지에 적합한가요?

단색 배경의 로고, 아이콘, 픽토그램, 단순한 일러스트레이션 등에 가장 적합합니다. 복잡한 사진이나 화려한 그라데이션이 들어간 이미지는 변환 품질이 떨어지거나 형태가 깨질 수 있습니다.

🔒 API 키는 안전한가요?

네, 입력하신 API 키는 외부 서버나 데이터베이스로 절대 전송되지 않습니다. 현재 사용 중인 브라우저의 로컬 스토리지에만 암호화되어 저장되므로 안심하고 사용하셔도 됩니다.
